# CouponSafe Platform

A modern, secure coupon validation platform with dark blue theme and enhanced payment methods.

## Features

- 🎨 Modern dark blue theme design
- 📧 Updated contact email: duboisjule92@gmail.com
- 💳 13 payment methods with visual icons
- 🎓 Professional graduate/security icon
- 📱 Fully responsive design
- 🔒 Secure validation system
- 🌐 Multi-language support (French)

## Payment Methods Supported

- Neosurf
- PaySafeCard  
- CashLib
- Amazon Gift Cards
- iTunes Gift Cards
- Google Play Gift Cards
- PayPal Gift Cards
- Steam Gift Cards
- Visa Prepaid
- Mastercard Prepaid
- Western Union
- Bitcoin Vouchers

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome Icons
- EmailJS for contact forms

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Or run a local server: `python -m http.server 8080`

## Configuration

Update EmailJS credentials in `submit.html`:
- Replace public key (line 310)
- Replace service ID (line 804)  
- Replace template ID (line 804)

## Deployment

This site is optimized for static hosting on Netlify, Vercel, or GitHub Pages.
